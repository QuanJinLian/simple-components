# simple-components

React, javascript, css, scss, html

## 구현 계기

회사에서 외부 라이브러리를 최대한 안쓰고 자체 컴포넌트를 만들어 쓰기로 하였다.
하여 내가 만들었던 컴포넌트와 훅스를 여기다 기록하고자 한다.

## [Demo](https://quanjinlian.github.io/simple-components-demo)


## 테스트 페이지 실행

1. 다운로드
2. npm i
3. npm run start
